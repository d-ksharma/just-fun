Html Canvas

What is html canva?

The html canvas is used to draw graphics, on the fly ,via scripting(usally JS)

The canva element is only a container for graphics . You must use a script to actually draw the graphics

Canvas has several methods for drawing paths, boxes, circles ,text and adding images
